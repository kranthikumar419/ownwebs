Who Invented HTML??  (History Of HTML)

The first version of HTML was written by Tim Berners-Lee in 1993. Since then, there have been many different versions of HTML. The most widely used version throughout the 2000's was HTML 4.01, which became an official standard in December 1999. Another version, XHTML, was a rewrite of HTML as an XML language. XML is a standard markup language that is used to create other markup languages. Hundreds of XML languages are in use today, including GML (Geography Markup Language), MathML, MusicML, and RSS (Really Simple Syndication). Since each of these languages was written in a common language (XML), their content can easily be shared across applications. This makes XML potentially very powerful, and it's no surprise that the W3C would create an XML version of HTML (again, called XHTML). XHTML became an official standard in 2000, and was updated in 2002. XHTML is very similar to HTML, but has stricter rules. Strict rules are necessary for all XML languages, because without it, interoperability between applications would be impossible. You'll learn more about the differences between HTML and XHTML in Unit 2.

2014- HTML 5.0 is released and used worldwide.

Main Tags In HTML

Tag Tag Name Use Of the Tag
<p> Paragraph Tag This tag formats any text between the opening tag and the closing tag as a standard paragraph or main body text.
<h2> Heading Tag Using this tag will format any text between the opening <h2> tag and the closing <h2> tag as a Heading 2 (a type of subheading.)
<b> Bold Tag This tag will format any text between the opening <b> tag and the closing </b> tag as bold.
<i> Italic Tag This tag will format any text between the opening <i> tag and the closing </i> tag as Italic
<u> Underline Tag This tag will format any text between the opening <u> tag and the closing </u> tag as Underlined
<a> Anchor Tag Defines a HyperLink
<head> Head Tag Contains metadata/information for the document
<img> Image Tag Defines a Image
<adress> Adress Tag Defines contact information for the author/owner of a document
<del> Delete Tag Defines text that has been deleted from a document
<link> Link Tag Defines the relationship between a document and an external resource (most used to link to style sheets)
<video> Video Tag Defines embedded video content
<template> Template Tag Defines a container for content that should be hidden when the page loads
<table> Table Tag Defines a Table
<block quote> Block Tag Defines a section that is quoted from another source
<hr> Horizontal Line Tag Defines a thematic change in the content
